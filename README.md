# Teslo API

1. **Intall dependencies.**

   Navigate into your new site’s directory and start it up.

   ```shell
   cd teslo-shop/
   yarn install
   ```
2. **Clone `.env.template`, rename to `.env` and assign respective values ​​to it.**

3. **Up postgresDB with docker.**

   ```shell
   cd teslo-shop/
   docker-compose up -d
   ```
4. **Run development mode:**

   ```shell
   cd teslo-shop/
   yarn start:dev
   ```
